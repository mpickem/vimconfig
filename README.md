## ViM configuration

#### Leader shortcuts

###### EasyMotion

**Shortcut** | **EasyMotion command**
:----: | ----:
s | single key search
d | double key search
h/l | inline movement


###### GitGutter

**Shortcut** | **GitGutter command**
:----: | ----:
j | next hunk
J | last hunk
k | prev hunk
K | first hunk
&lt | stage Hunk
&gt | undo Hunk


###### FZF
**Shortcut** | **FZF command**
:----: | ----:
a | **A**g
b | **B**uffers
f | **F**iles
g | **G**it files
m | **M**arks
t | **T**ags
space | Lines in the current buffer


* w :: toggle **W**hitespace highlighting

#### Control shortcuts
* n :: **N**erdtree
* l :: remove highlighting after search

#### General mappings
* Y :: y$
* F1 :: FZF Helptags
* F2 :: pastetoggle
* F3 :: toggle line numbers
* Q :: Nope
* space :: fold/unfoldswp
